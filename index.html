O Hackathon

É com muita animação que o MATRIX Labs Brasil em parceria com a AbsamHost e do Diretório Acadêmico do curso de Engenharia de Computação da UEFS anuncia o MATRIX Labs Hackathon durante a XVII SIECOMP. O MATRIX Labs Hackathon irá acontecer dentro da XVII SIECOMP nos dias 16 e 17 de Março.

As inscrições estão abertas à todos os participantes da XVII SIECOMP e devem ser feitas através do nosso formulário de inscrição. Durante todo o hackathon o time do MATRIX Labs Brasil e seus parceiros estarão dando apoio técnico, mentoria e ideação.

Desafio: Construa um app para o MATRIX Creator que ajude a melhorar o dia-a-dia das pessoas no qual use pelo menos três recursos da API do MATRIX Cretor e uma API de qualquer serviço externo de sua escolha.

Prêmio: O time vencedor receberá 1 kit MATRIX Creator e R$ 500 reais de servidor na AbsamHost. O segundo lugar receberá 1 kit MATRIX Creator. O terceiro colocado receberá R$ 250 reais de servidor na AbsamHost. Cada kit MATRIX Creator conta com: MATRIX Creator + Raspberry Pi 3 + Raspberry Pi Câmera V2 + Cartão de Memória de 32GB + Fonte de Alimentação.

Regras

Todas as equipes devem ter 4 pessoas.

Todas as equipes devem mandar seus times para o email (diego.leite@admobilize.com), para poder participar do evento. O título deve ser o nome do projeto. O corpo do e-mail deve conter uma descrição de até 140 caracteres sobre o projeto e o nome de cada integrante do time.

Para receber o kit MATRIX Creator de trabalho cada integrante do time deverá assinar o termo de responsabilidade que será fornecido no dia do hackathon.

Para que a equipe possa iniciar os trabalhos deve ser criado um repositório aberto no github. O link deve ser enviado na mesma thread de e-mail até o horário limite descrito na agenda do hackathon.

Para que o projeto possa ser apresentado na sessão de pitches o app desenvolvido deve ter sido publicado na MATRIX App Store. O link gerado deve ser enviado na mesma thread de e-mail até o horário descrito na agenda do hackathon.

Todas as equipes convocadas para a sessão de pitch terá que apresentar uma demonstração de sua solução e slides não serão permitidos. A apresentação será de 5 minutos + 3 minutos de sessão de perguntas e respostas dos jurados.

As equipes vencedoras estarão sujeitas a uma revisão de código em algum momento após o evento, ou imediatamente antes de receber o prêmio.

É proibido o uso de código pronto! Todos iniciarão a maratona de programação ao mesmo tempo. Isso garante a imparcialidade da competição. Não participe da competição se for reutilizar um projeto já existente.

Projetos

Moodify
Team: Pedro Cardoso, Luan Rios, Rejanio Moraes
Description: Recognize user's mood, search for Spotify playlists to help his day. These playlists are found by user's interest, crossing information like geolocation and most played songs.

Maternity Sentinel
Team: João Victor Mota, Lucas Carneiro, Lucas Vinicius, Nilson Augusto
Description: A device responsible by monitoring maternity environment. The main features are: Facial recognition for medical responsible team, verification for humidity and temperature to certify the ideal conditions for maternity.

PillController
Team: Weverson Gomes, José Victor, Roberto Nunes, Iago Rios
Description: The main purpose is control lockers that keeps pills and helps users not forget to take his medicines.

Slider
Team: Victor R. Figueiredo, Lucas A. Silva, Leandro Sampaio, Matheus Galvão
Description: Control Google Slide presentations with gestures.

SmartOffice
Team: Lucas Paiva, Francisco Alberto, Lindelmo Silva, Cleybson Leite
Description: Automate an office, you will be able to control room temperature and light using the smartphone. Another great feature is the ability to unlock the office door using facial recognition.
